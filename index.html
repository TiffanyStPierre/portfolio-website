Pet Adoption App

React, Firebase, CSS, Javascript

App for users to list and browse pets for adoption. Has Firebase user authentication and database.

Click here to view a video walkthrough of the app

Click here to view the GitHub repository

This pet adoption app was built using React, React Router & Firebase. The app allows users to log in and create, edit and delete listings. It also allows users to browse pet listings by category and view the detailed listing information. There is a contact form that will generate an email template to be sent to the owner of the listing.

Trivia Game

React, CSS, Javascript, Fetch API

Trivia app with various question categories. The game tracks the player's score in state.

Click here to view the deployed app

Click here to view the GitHub repository

This trivia app was built using React & React Router. The trivia questions are fetched from a third party API, Open Trivia Database. The app uses state, held in context, to track the player's score until the game is reset. The app is intentionally styled with a retro feel.

Movie Finder App

HTML, CSS, Javascript, Fetch API

App that recommends movies & TV shows to the user, with data from an external API.

Click here to view the deployed app

Click here to view the GitHub repository

This movie finder app uses an external API call to The Movie DB to fetch lists of the top 20 trending movies and TV shows of the week. The user can click on either the movie or TV button to get a random recommendation from the selected top 20 list. This app uses Netlify serverless functions to conceal the API key.
