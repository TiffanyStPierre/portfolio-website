Project Details

I enjoy keeping up to date on the newest movies and TV shows so I decided to create a movie and TV show recommendation app. It uses an external API call to The Movie DB to fetch lists of the top 20 trending movies and TV shows of the week when the app is loaded on screen. When the user clicks on either the TV or Movie button, it displays the details of a random selection from the selected list (movie or TV). I opted to build this app with HTML, CSS and vanilla Javascript.

I used Figma to create a design for the app and then went ahead and started building it. I quickly realized that I needed to find a way to conceal my API key from the browser, which is something I hadn't initially considered and didn't know how to do. I did my research and discovered that Netlify serverless functions and environment variables would work to hide my API key. After reading over the documentation, watching a couple of videos and some trial and error, I was able to figure out how to successfully use these Netlify features for my app.

I learned a lot more than I expected to during the process of building this app. Not only did I learn about Netlify serverless functions and environment variables, I also learned how to deploy directly to Netlify from GitHub and how to use the Netlify CLI.
