Project Details

I built this trivia game using React and React Router. The trivia questions are fetched from a third party API, Open Trivia Database. The app has 3 pages - a landing page, an error page and a quiz page.

When the user starts the game, they can select a trivia category and the app will return 4 multiple choice questions from that category. When the user selects an answer for each question, their answer will be noted and the correct answer will also be shown. Each question reveals the correct answer separately. The user can reset the game or get questions from another category at any time.

This is the first React app I built and deployed completely on my own from scratch. I had some struggles with the quiz cards component, specifically because the API returned HTML special characters. I had to learn how to use dangerouslySetInnerHTML to reformat the special characters so they would display in a readable format in both the questions and answers displayed in the DOM.

This was a really great beginner project to practice my React skills and I believe it gave me a good foundation on which to build upon for future projects.
